# Password Checker

Программа проверяет пароль по нескольким критериям:
- длина не менее 8 символов
- наличие цифры
- наличие специального символа

## Возможности
- проверка длины пароля
- проверка цифр
- проверка специальных символов
- программа работает в цикле
- обработка ошибок ввода

## Как запустить

1. Скачать файл password_checker.py
2. Открыть терминал
3. Запустить команду:

python password_checker.py

## Пример работы

Введите пароль: hello123

Проверка пароля:
✘ Нет специального символа

Введите пароль: Hello123!

Проверка пароля:
✔ Длина не меньше 8 символов
✔ Есть цифра
✔ Есть специальный символ
